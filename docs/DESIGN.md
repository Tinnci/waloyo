# Waloyo UI/UX 设计指南

本文档定义 Waloyo 应用的视觉设计语言和交互模式。

## 设计理念：Wind & Rain (风与雨)

Waloyo 的核心设计隐喻源自自然界的风雨现象：

### 隐喻映射

| 状态 | 自然隐喻 | 视觉表现 |
|------|----------|----------|
| 待办任务 | 风 (Wind) | 动态、不安定、轻微摇摆 |
| 完成动作 | 雨 (Rain) | 下落、液化、消散 |
| 全部完成 | 晴空 (Clear Sky) | 明亮、平静、庆祝 |

## 间距系统

使用一致的间距单位：

| 名称 | 值 | 用途 |
|------|-----|------|
| PADDING_SM | 8px | 元素内部小间距 |
| PADDING_MD | 16px | 标准间距 |
| PADDING_LG | 24px | 区域间的大间距 |

## 圆角系统

| 名称 | 值 | 用途 |
|------|-----|------|
| RADIUS_SM | 4px | 小元素 |
| RADIUS_MD | 8px | 卡片、按钮 |
| RADIUS_LG | 12px | 大型容器 |

## 动画系统

### 时长规范

| 名称 | 值 | 用途 |
|------|-----|------|
| ANIM_FAST | 150ms | 微交互（hover, focus） |
| ANIM_NORMAL | 300ms | 标准过渡 |
| ANIM_SLOW | 600ms | 复杂动画 |
| ANIM_RAIN_DROP | 800ms | 雨滴下落效果 |

### 动画效果规划

#### 1. Rain Drop (雨滴下落) - 完成任务

```
点击瞬间
    │
    ▼ 0ms
┌─────────────────┐
│ 颜色变为雨蓝    │
│ 开始聚集效果    │
└────────┬────────┘
         │
         ▼ 200ms
┌─────────────────┐
│ 开始下落        │
│ Y轴加速位移     │
│ 轻微水平摇摆    │
└────────┬────────┘
         │
         ▼ 600ms
┌─────────────────┐
│ 透明度渐变      │
│ 缩放变小        │
└────────┬────────┘
         │
         ▼ 800ms
┌─────────────────┐
│ 从列表移除      │
│ 加入已完成区    │
└─────────────────┘
```

#### 2. Wind Sway (风中摇曳) - 待办任务

- 轻微的左右摇摆
- 周期性动画 (2-3秒一个周期)
- 不同任务有轻微相位差

#### 3. Clear Sky Celebration (晴空庆祝)

- 背景颜色渐变 (storm → clear)
- 标题颜色变为金色
- 可选：粒子效果（阳光）

## 交互设计

### 点击反馈

1. Hover: 背景色变亮
2. Active: 轻微按压效果
3. Click: 触发完成动画

### 列表滚动

- 平滑滚动
- 弹性边界
- （未来）虚拟化长列表

## 响应式考虑

当前设计针对桌面端 (420x680)，未来可扩展：

- 窄屏幕：减少水平边距
- 宽屏幕：限制最大宽度，居中显示

## 无障碍考虑

1. 颜色对比度符合 WCAG AA 标准
2. 所有交互元素有清晰的焦点状态
3. 动画可通过系统偏好减少
